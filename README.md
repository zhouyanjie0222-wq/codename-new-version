# English Vocab Codenames (Standalone Edition)

This is a **no-build** version of the Codenames game, specifically designed for easy deployment to GitHub Pages. 

## 🚀 How to Deploy to GitHub Pages

1. **Export to GitHub**: Use the "Export to GitHub" button in AI Studio.
2. **Enable Hosting**:
   - Go to your repository **Settings** > **Pages**.
   - Under **Build and deployment** > **Source**, keep it as **"Deploy from a branch"**.
   - Select the **main** branch and the root **/** folder.
   - Click **Save**.
3. **Wait 1 minute**: Your site will be live at `https://your-username.github.io/your-repo-name/`.

## 🤖 Activating AI Definitions
To enable the AI definition feature in this version:
1. Open `index.html` in your text editor.
2. Search for `GEMINI_API_KEY_PLACEHOLDER`.
3. Replace it with your actual Gemini API key (keep the quotes around it).
4. Save and push to GitHub.

## Features
- **Standalone**: All code is in one file (`index.html`).
- **No Build Required**: Just open the file in any browser.
- **AI Integration**: Powered by Google Gemini.
- **Polished Design**: Styled with Tailwind CSS and Lucide icons.
